CURRENEX

Currency Converter Application

A Python-based currency converter application utilizing real-time exchange rates API. Features include:

  • Modern UI built with customtkinter library
  • Real-time currency conversion
  • Support for multiple currencies
  • View Project

Sparty Boots

A C++ application where a character (a.k.a Sparty) kicks a products of a conveyor based on certain conditions. To determine whether to kick the product, the program asks the user to create a digital logic circuit.

Sparty in the Box

A C++ application creating a jack in the box using Sparty as a character. The box was then adapted into another project which was the Canadian Experience. The Canadian Experience allows keyframes for animation to support a movie involving Sparty, Harold and the jack in the box machines.

Hangman Application

A Python Tkinter Hangman application getting words using the Words API

ROS2 Convoy Simulation

Created an autonomous convoy using ROS2. A custom world was created in Gazebo Simulator using two turtlebot models: Burger and Waffle. Waffle was the leader and Burger was the follower. Waffle was moved using a python package and the follower tried moving closer when the distance between them got higher than a threshold distance.
